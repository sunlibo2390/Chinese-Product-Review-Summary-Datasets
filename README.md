# Product-CSum-Cross

共包含化妆品、汽车、餐厅三个品类

**表格字段**

- Name：商品品类
- Text：单条评论或多条评论（用‘|||’分隔）
- Aspects：用gpt3.5-turbo对text抽取得到的“角度-极性-对应句子”
- Summary_list：针对该text的归纳
- Summary_num：归纳数量

| ------ | ------------ | --------------- |
